Install puriKRON app

Add to your home screen for quick access — same pattern as entelekron.app and app.sovraprotocol.com.

iPhone / iPad (Safari):

  1. Tap the Share button (square with arrow)
  2. Scroll down → Add to Home Screen
  3. Tap Add

iOS never shows an automatic install popup — this manual step is required.

Android (Chrome):

  1. Tap Install App below, or
  2. Menu (⋮) → Install app or Add to Home screen

Desktop: Click the install icon in the Chrome/Edge address bar, or bookmark this page.
