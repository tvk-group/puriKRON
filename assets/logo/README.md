# puriKRON Logo Set

Official puriKRON brand assets derived from the uploaded master logo.

## Source file (master)

Replace this file with your exact logo export to rebuild everything:

**[`purikron-logo-source.png`](./purikron-logo-source.png)**

Then run:

```bash
python3 assets/logo/build-from-source.py
```

## Quick download — 32×32 SVG icon

[Download `icon-32.svg`](https://raw.githubusercontent.com/tvk-group/puriKRON/main/assets/logo/icon-32.svg)

```text
https://raw.githubusercontent.com/tvk-group/puriKRON/main/assets/logo/icon-32.svg
```

The SVG embeds the exact 32×32 PNG raster from the master logo (pixel-accurate, not hand-drawn).
